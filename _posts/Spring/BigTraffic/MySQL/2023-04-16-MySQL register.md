# 회원정보 등록 구현 with MySQL
MySQL 맛보기 (2)

## 회원정보 관리
- 이메일, 닉네임, 생년월일을 입력받아 저장한다.
- 닉네임은 10자를 초과할 수 없다.
- 회원은 닉네임을 변결할 수 있다.
	- 회원의 닉네임 변경이력을 조회 하 수 있어야한다.
