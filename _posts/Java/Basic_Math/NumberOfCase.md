# 경우의 수

- 어떤 사건에서 일어날 수 있는 경우의 가짓수
	- 예) 동전을 던지는 사건 -> 경우의 수 2
	- 예) 주사위를 던지는 사건 -> 경우의 수 6

```
사건 A가 일어날 경우의 수 : n(A)
```
