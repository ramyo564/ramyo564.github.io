## JavaScript

```javascript
// 내가 푼 답
function solution(age) {
    var answer = 2023-age;
    return answer;
}
// 다른 사람 풀이
function solution(age) {
    return new Date().getFullYear() - age + 1;
}
```
### 정리
- Date().getFullYear() 는 현재 년도를 갖고옴
